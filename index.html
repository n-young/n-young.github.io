Projects

LinguaML is a computer-vision powered language learning app designed for on-the-go learners. With just a picture of your subject, flashcards for any language are generated within seconds and saved in the app. You can view, study, edit, and share flashcards with your friends!

LinguaML was conceived and developed at Hack@Brown 2020. It won the Facebook award for "Best hack that builds community through private or small-group interaction".

Repo React Native Node.js Google Vision API

ParselTongue is a paper parser that helps researchers and academics skim through dense documents quickly. By uploading your PDF and specifying key search words, ParselTongue gives you only the most salient parts of the paper, saving hours of time!

ParselTongue was developed as a final project for CSCI 0320 - Software Engineering at Brown University. It relies on the PageRank algorithm.

Repo Demo Java Javascript HTML/CSS

Dear Blueno Bot is an automatic "Dear Blueno" post generator developed with a TensorFlow natural language processing library. The data is collected from two source files, provided by the DB mods, and was parsed through Beautiful Soup.

Repo Demo Python Tensorflow Flask

CookieCash helps parents motivate their kids by offering a medium for incentives! Parents can set rewards on certain activities or goals for the child to complete, and the child can redeem these awards for more parent-defined prizes.

CookieCash was conceived and developed at YHack 2019.

Repo React Node.js Express
